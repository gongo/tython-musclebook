Tython MuscleBook
=================

## make book

### Requirements

- bash
- [melborne/maliq](https://github.com/melborne/maliq)
  - `$ gem install maliq`
  - When use [bundlizer](https://github.com/Tomohiro/bundlizer), `$ bundlizer install maliq`

### build

    $ cd /path/to/tython-musclebook
    $ ./tools/gen_epub

make `build/out.epub`
